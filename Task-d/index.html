In CSS, selectors declare which part of the markup a style applies to by matching tags and attributes in the markup itself.

What Are CSS Selectors?

CSS selectors are patterns used to target HTML elements. They tell the browser which elements to style and how.

Element Selectors

Element selectors apply styles directly to HTML elements like p, h1, or div.
